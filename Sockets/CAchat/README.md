# CAchat

This respositories contains examples of node js with socket.io. In this case, the example is a web chat.

::::::::: Download and play

## Install environment
```
$ npm install
```

## Run app
```
$ hotnode server.js //for dev
```
or 
```
$ node server.js
```
## Go to
http:localhost:2018 --> voalá :+1:
